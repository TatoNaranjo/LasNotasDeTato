Se conoce como la conversión de un elemento dentro de un conjunto a una variable difusa.

![[Pasted image 20250926094220.png]]

Para poder lograr esto se deben de utilizar algunas funciones de membresía que denotan una calidad, una precisión y una finura. Las funciones `f(x)` más utilizadas son: `Función triangular` y `Función Trapezoidal`.

La función triangular está representada por dos intervalos: {A,B} representando una recta creciente positiva, y {B,C} representando una recta decreciente negativa.

La función trapezoidal está representada por 3 intervalos: {A,B} representando una recta creciente positiva, {B,C} representando un valle, y {CD} representando una recta decreciente negativa.
